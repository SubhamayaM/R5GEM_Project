🚄 R5GEM: AI-Based Stable 5G System for Railways

A simulation-driven AI system that analyzes railway network coverage, predicts signal strength using machine learning, and compares traditional communication models with an intelligent multi-connectivity approach to ensure stable connectivity for high-speed trains.

🚀 Features

📍 Interactive route selection on India map (start → end)

📡 Tower-based coverage analysis using real-world telecom data

🤖 AI-powered signal prediction (ML model)

🔄 Dynamic nearest tower selection (handover simulation)

📊 Model comparison (Baseline vs Proposed system)

📉 Performance evaluation (Throughput, Latency, Packet Loss)

🗺️ Visual coverage mapping (Excellent / Good / Poor / No Signal)

💾 Saves results and graphs in results/ folder
